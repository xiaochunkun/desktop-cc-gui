## 2. 结构化变更清单（主题级）

### 2.1 功能新增

- 幕布宽度模式（`canvasWidthMode`: `narrow|wide`）与非法值回退。
- 左右布局切换（`layoutMode`: `default|swapped`）与 Win/mac 兼容。
- 侧栏会话/看板快捷入口增强与 swapped 镜像布局补齐。
- 会话列表体积展示（`sizeBytes`）与分级视觉样式。
- `File changes` 多文件逐行折叠/独立展开。
- MCP 设置页重构为“按引擎查看”的只读总览视图。
- Usage 多引擎统计增强（engine distribution、AI modified lines、daily code changes）。
- 品牌升级为 `ccgui` + legacy 数据迁移。

### 2.2 修复与稳定性

- 启动链异常黑屏兜底页面。
- Web 切换 Codex 后 `workspace not connected` 场景重连与重试修复。
- 本地扫描不可用时保留已知 Codex 会话并打 `partialSource` 标记。
- Claude 会话销毁期间子进程竞争/残留修复。
- 默认 workspace 去重与路径边界修复。
- swapped 模式快捷入口顺序与快捷键显示修复。
- Windows 文件树拖拽反馈修复。
- Composer 长文本溢出与实时更新干扰修复。

### 2.3 重构/治理

- Claude 事件转换模块拆分（大文件治理，无行为变更）。
- settings 样式文件拆分（大文件治理）。

### 2.4 文档/发布类

- changelog 增量补齐（中英文）。
- 版本号 bump 到 `0.3.12`。
