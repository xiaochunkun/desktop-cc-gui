## 2. 结构化变更清单（主题级）

### 2.1 功能新增

- 侧栏根线程支持悬停图钉、固定区支持一键取消固定。
- Git 提交信息支持语言选择（中文/英文）与引擎选择（Codex/Claude/Gemini/OpenCode）。
- Composer 新增独立“快捷动作”图标入口与二级菜单。
- 用户消息支持 `@路径` 引用提取并以独立引用卡片展示。
- Codex/Claude/Gemini 新增流式 `waiting/ingress` 相位视觉联动。
- `v0.3.12` 新增 `canvasWidthMode`、`layoutMode`、会话大小展示、MCP 引擎只读视图、Usage 多引擎统计增强。

### 2.2 修复与稳定性

- Gemini 会话续传、pending 关联、长 prompt stdin 传输、Windows 截图路径兼容修复。
- Claude 子进程终止竞争/残留、初始化失败残留、自定义命令空响应回退修复。
- 多会话 stop 误伤、跨会话绑定边界、Web 切 Codex 后无法继续对话等修复。
- `@路径` 引用在相邻文本场景的解析修复与引用卡片视觉层级修复。
- swapped 布局快捷入口顺序/快捷键文案修复，侧栏折叠布局与图标反馈修复。

### 2.3 重构与工程治理

- Claude 事件转换模块拆分（大文件治理）。
- Settings 样式模块拆分与基础外观模块化。
- 多处测试补齐（消息卡片可见性、快捷动作、流式相位等）。

### 2.4 配置、品牌与发布

- 品牌切换为 `ccgui` 并提供 legacy 数据迁移。
- `0.3.9~0.3.12` 版本 bump 与 changelog 中英文同步。
